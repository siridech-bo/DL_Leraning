# Lesson 6: Collaborative Filtering

## Summary
- Build a simple recommender using embeddings for users and items.

## Key Points
- Dataloaders for user‑item interactions.
- Embedding dimensions and regularization matter.
- Evaluate with ranking metrics or RMSE depending on formulation.

## Tasks
- Create a small interaction dataset.
- Train, evaluate, and inspect sample recommendations.
- Discuss cold‑start considerations.
