# Lesson 4: NLP Classification

## Summary
- Train a simple text classifier using fastai text dataloaders.
- Explore tokenization, vocabulary, and sequence length tradeoffs.

## Key Points
- Preprocessing: tokenization and numericalization.
- Model: transfer learning with a pretrained language model head.
- Metrics: accuracy/F1; handle class imbalance if present.

## Tasks
- Load a small text dataset with labels.
- Train, evaluate, and inspect misclassifications.
- Experiment with sequence length and minimal text cleanup.

## Resources
- IMDb dataset `https://ai.stanford.edu/~amaas/data/sentiment/`
- AG News (alternative) `https://www.kaggle.com/datasets/amananandrai/ag-news-classification-dataset`
- fastai text `https://docs.fast.ai/text.html`
